## Descripción

Este repositorio contiene el desarrollo progresivo de prácticas, ejercicios, ejemplos y proyectos realizados durante la materia de Programación y Plataformas Web utilizando Angular.

El objetivo del repositorio es documentar el aprendizaje y la evolución de cada módulo, manteniendo una estructura organizada y escalable para futuras implementaciones.

Aquí se almacenarán:

- Prácticas de Angular
- Ejercicios de TypeScript
- Configuración de proyectos
- Routing y navegación
- Componentes
- Servicios
- Formularios
- Consumo de APIs
- Manejo de estado
- Estilos y diseño
- Evidencias y capturas
- Proyectos incrementales

---

## Objetivos del Repositorio

- Centralizar todas las prácticas desarrolladas con Angular.
- Mantener una estructura organizada por módulos o temas.
- Aplicar buenas prácticas de desarrollo frontend.
- Documentar el progreso del aprendizaje.
- Utilizar control de versiones mediante Git y GitHub.
